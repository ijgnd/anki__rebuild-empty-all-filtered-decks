An Anki add-on that adds "Rebuild All" and "Empty All" buttons to the bottom of the main screen, to rebuild or empty all filtered decks at once. The original for Anki 2.0 is here. khonkhortisan released it for 2.1 as Rebuild All & Empty All for Anki 2.1. My release adds the option to set in the add-on config which decks are built first and last: This might be useful if a card matches the criteria of multiple of your filtered decks because cards can only be in one filtered deck. # bugs, errors, feedback Please do not use reviews for bug reports or support requests.
